Sistema de Avaliação de Estudantes – Projeto em C

Este é o primeiro projeto acadêmico que eu, junto com mais 2 colegas, fizemos. Desenvolvido em linguagem C, tem como objetivo criar um sistema simples para cadastrar estudantes, receber suas notas e calcular automaticamente a média final. O programa considera as três avaliações (AV1, AV2 e AV3), ordena as notas e utiliza apenas as duas maiores para compor a média — seguindo um critério comum de avaliação.

Ao final, o sistema exibe o nome do aluno, sua média calculada e o status de Aprovado ou Reprovado, com base em uma média mínima pré-definida.

Funcionalidades
-
Cadastro do nome de até cinco estudantes.

Entrada das notas de três avaliações (AV1, AV2 e AV3).

Ordenação das notas em ordem crescente para selecionar as duas maiores.

Cálculo automático da média final.

Exibição do resultado: Aprovado ou Reprovado.

Estrutura organizada usando struct para representar cada estudante.

Tecnologias e Recursos Utilizados
-
Linguagem C

Estruturas (struct)

Manipulação de strings

Ordenação simples de vetores

Entrada e saída padrão (stdio.h)
